# What

数据结构( data structures )指的是一组数据的存储结构（数组、队列、链表、二叉树、... ）。

算法( algorithms )就是操作数据结构的一组方法（递归、排序、查找、搜索、... ）。

数据结构和算法解决的是如何更省、更快地存储和处理数据的问题。复杂度分析( complexity analysis )是对应的一个考量方法。

# How
如何学习数据结构与算法：
* 动手写
* 写出性能更好的代码
* 锻炼解决问题的思路和方法
